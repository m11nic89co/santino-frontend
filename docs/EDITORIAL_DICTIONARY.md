# Редакционный стандарт SANTINO (эталон на 1 страницу)

**Назначение:** единые правила текстов на сайте, в CTA, контактах и метаданных.  
**Область:** публичные тексты `santino-frontend`, согласованные формулировки с [santino.market](https://santino.market/) при необходимости.

---

## 1. Тон и обращение

| Правило | Стандарт |
|--------|----------|
| Стиль | Деловой, ясный, без канцелярита и без «воды». |
| Обращение | **Везде одинаково:** либо `вы / ваш / вам` (строчная), либо `Вы / Ваш / Вам` (прописная). Рекомендация для B2B: **строчная**. |
| Обещания | Конкретика: сроки, этапы, контроль качества — без голых прилагательных без опоры. |

---

## 2. Глоссарий (термины)

| Использовать | Не использовать |
|--------------|-----------------|
| **прайс-лист** | прайс, Прайс, price-list в русском интерфейсе |
| **контрактное литьё** | «литьё по контракту» в разных формах в одном экране |
| **Рассчитать** (глагол) | Расчитать |
| **E-mail:** или **Эл. почта:** (выбрать **один** вариант на весь сайт) | Емайл, e-mail вперемешку |
| **координаты для навигатора** | кор-ты, коорд. |
| **техническое задание** / **ТЗ** (если уже введено в тексте) | смешение «ТЗ» и «техзадание» в одном блоке без пояснения |

**Орфография:** по словарю; спорные формы — по [Розенталю](https://old-rozental.ru/) или актуальной норме.

---

## 3. CTA (кнопки и ссылки)

Выбрать **одну** схему на весь продукт и держаться её:

**Схема A — глагол + объект**

- Запросить прайс-лист  
- Рассчитать выпуск изделия / Рассчитать контрактное литьё  

**Схема B — получить + объект**

- Получить прайс-лист  
- Получить расчёт выпуска изделия  

Правила:

- Регистр в кнопке **единообразный** (не смешивать «Запросить прайс-лист» и «ЗАПРОСИТЬ ПРАЙС» без дизайн-причины).
- Текст кнопки = действие, которое произойдёт после клика (или явная подпись рядом).

---

## 4. Контакты (шаблон подписей)

Единый блок подписей:

1. **Телефон:** или **Телефоны:** (один вариант на сайт)  
2. **E-mail:** *или* **Эл. почта:**  
3. **Адрес:**  
4. **Координаты для навигатора:** (при необходимости)

Адрес: либо полностью (**Московская область, Раменский район**), либо сокращённо (**Моск. обл., Раменский р-н**) — **один стиль** на все карточки.

---

## 5. Типографика (кратко)

| Элемент | Правило |
|---------|---------|
| Тире | Смысловая пауза — **длинное тире** `—` |
| Дефис | Только внутри слов и в «прайс-лист» |
| Кавычки в русском тексте | **«ёлочки»** |
| Слэш | `24/7` без пробелов вокруг слэша в таких маркерах |
| Капс | Только осознанно (заголовки секций по макету), не «кричать» в абзацах |

---

## 6. SEO и мета

- `title`, `description`, `og:*` — **тот же тон**, что и на странице; без новых терминов, которых нет в интерфейсе.
- Ключевые слова из глоссария совпадают с текстом на экране.

---

## 7. Быстрая проверка перед публикацией

- [ ] Нет `Расчитать`, `Емайл`, `кор-ты`.  
- [ ] Везде один выбор: `E-mail` **или** `Эл. почта`.  
- [ ] `прайс-лист` написан через дефис, стиль CTA единый.  
- [ ] Обращение к читателю в одном регистре по всему сайту.  
- [ ] Контакты и юр. блоки — один шаблон подписей.
