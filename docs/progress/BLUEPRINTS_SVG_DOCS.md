# 📐 Система фоновых чертежей на SVG.js

## 🚀 Quick Start

### Установлено:
```bash
npm install @svgdotjs/svg.js
```

### Подключено:
```html
<script type="module" src="assets/js/modules/ui/animations/blueprints-svg.js"></script>
```

### Результат:
Профессиональные чертежи горшков появляются на фоне Hero секции.

---

## 📋 Типы чертежей

### 1. Цилиндрический горшок (Cylindrical)
- **Форма:** Цилиндр с квадратными стенками
- **Размеры:** Ø180мм × 200мм
- **Элементы:** Стенки, вид сверху (окружность), линии размеров

### 2. Квадратный горшок (Square)
- **Форма:** Квадратный горшок
- **Размеры:** 150×150мм × 180мм
- **Элементы:** Контур, 3D вид, диагонали

### 3. Круглый горшок (Round)
- **Форма:** Широкий цилиндр
- **Размеры:** Ø200мм × 150мм
- **Элементы:** Две окружности, стенки

### 4. Сужающийся горшок (Tapered)
- **Форма:** Трапеция/конус
- **Размеры:** Ø160мм × 220мм
- **Элементы:** Трапециевидный контур, вид сверху и снизу

---

## 🎨 Стили чертежей

### Основные линии:
- Цвет: `#ffffff`
- Толщина: `2.5px`
- Скругление: `linecap: 'round'`

### Линии размеров:
- Цвет: `#ffffff`
- Толщина: `1.5px`
- Прозрачность: `0.7`

### Пунктирные линии:
- Вид сверху/снизу: `stroke-dasharray: '4,2'`
- Стиль: AutoCAD-like

---

## ⚙️ Конфигурация

### Параметры системы:
```javascript
maxBlueprints = 3;              // Макс. чертежей
firstDelay = 500ms;             // Задержка первого
interval = 4000-7000ms;         // Интервал создания
lifetime = 10000-15000ms;       // Время жизни
scale = 0.8-1.3;                // Размер
rotation = -20 to +20 deg;      // Поворот
```

## 📐 Примеры чертежей

### Цилиндрический:
```
      ○ (вид сверху - пунктир)
    ┌─────┐
    │     │  (стенки)
    │     │
    └─────┘
════════════════ (линии размеров)
```

### Квадратный:
```
      ▢ (вид сверху - пунктир)
    ┌─────┐
    │     │  (стенки)
    │     │
    └─────┘
════════════════ (линии размеров)
```

## 🔧 API

### Публичные методы:

```javascript
// Получение экземпляра
const system = window.BlueprintsSVG;

// Создание чертежа вручную
system.createBlueprint('cylindrical');

// Уничтожение системы
system.destroy();
```
