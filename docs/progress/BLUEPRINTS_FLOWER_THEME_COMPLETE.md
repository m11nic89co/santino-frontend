## ✅ НОВЫЕ ЧЕРТЕЖИ НА ЦВЕТОЧНУЮ ТЕМАТИКУ:

### 1. RibbonPot ✅
**Наименование:** Горшок с декоративным ободком (лента)  
**Размеры:** Ø165мм × 180мм  
**Особенности:**
- Декоративный ободок по периметру
- Волнистый верхний край
- ГОСТ штамп
- Центровые линии
- Размерные линии с стрелками
- Штриховка 45°

### 2. FlowerPot ✅
**Наименование:** Цветочный горшок с лепестками  
**Размеры:** Ø190мм × 210мм  
**Особенности:**
- 5 лепестков по окружности
- Концентрические окружности
- ГОСТ штамп
- Штриховка по лепесткам
- Размеры с выходом за контур

### 3. TulipPot ✅
**Наименование:** Горшок "Тюльпан"  
**Размеры:** Ø180мм × 200мм  
**Особенности:**
- Форма расширяется к низу (как тюльпан)
- Плавные линии
- 3 яруса структуры
- ГОСТ штамп
- Штриховка наклонная

### 4. RosePot ✅
**Наименование:** Горшок "Роза"  
**Размеры:** Ø200мм × 230мм  
**Особенности:**
- Многоярусная структура
- 3 яруса с плавными переходами
- Лепестковая форма
- ГОСТ штамп
- Штриховка многослойная

### 5. SunflowerPot ✅
**Наименование:** Горшок "Подсолнух"  
**Размеры:** Ø210мм × 190мм  
**Особенности:**
- Круглый с резными краями
- 16 лепестков по периметру
- Концентрические окружности
- ГОСТ штамп
- Радиальная штриховка
