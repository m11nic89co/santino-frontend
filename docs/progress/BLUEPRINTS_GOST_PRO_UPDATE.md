# ✅ УСОВЕРШЕНСТВОВАНИЕ ЧЕРТЕЖЕЙ ДО ГОСТ ПРОФИ

## ✅ ПРИМЕНЕННЫЕ УЛУЧШЕНИЯ ГОСТ:

### 1. Штамп (ГОСТ 2.104-2006) ✅
- **Основная надпись** в правом нижнем углу
- **Формат:** А4
- **Масштаб:** 1:1
- **Наименование:** "ЧЕРТЕЖ"

### 2. Центровые линии (ГОСТ 2.303-68) ✅
- **Выходят за контур** на 3-5мм
- **Пунктир** 2,2 для центровых
- Горизонтальная и вертикальная линии

### 3. Осевая линия ✅
- **Толстая сплошная** для оси симметрии
- Проходит через весь чертеж

### 4. Размерные линии (ГОСТ 2.307-2011) ✅
- **Разрыв** 2-3мм между линиями и контуром
- **Стрелки:** размер 5px (увеличены с 4px)
- **Пунктирная** разрывная линия
- **Текст размеров** жирный, моношрифт

### 5. Штриховка сечений (ГОСТ 2.306-68) ✅
- **Угол 45°** (обязательно)
- **Шаг** 12-14мм
- **Тонкая линия** 0.8px

### 6. Линии по ГОСТ 2.303-68 ✅
- **Сплошная толстая** (3px) - видимый контур
- **Сплошная тонкая** (1.5px) - внутренние линии
- **Штрихпунктирная** - невидимый контур
- **Сплошная волнистая** - обрыв линии

### 7. Техническая информация (ГОСТ) ✅
- **Наименование** детали
- **Буквы** для разрезов (A, B, C)
- **Маркировка** видов (А-А, Б-Б)

### 8. Масштаб и габариты ✅
- **Масштаб:** 1:1 (ГОСТ 2.302-68)
- **Габаритные размеры** в мм
- Указаны на чертеже
