## ❌ ПРОБЛЕМА:

Виделись только 2-3 чертежа максимум вместо 5.

### Причины:
1. ❌ Интервал создания слишком большой (2-3.5 секунды)
2. ❌ Чертежи исчезали быстрее (5-8 секунд)
3. ❌ Не создавалось достаточно чертежей для поддержания 5 штук
4. ❌ Только 1 чертеж создавался при старте

---

## ✅ ИСПРАВЛЕНИЯ:

### 1. Интервал создания уменьшен ✅
- **Было:** 2000-3500ms
- **Стало:** 800-1200ms (каждые 0.8-1.2 секунды)
- Чертежи создаются в 2-3 раза чаще

### 2. Время жизни увеличено ✅
- **Было:** 5000-8000ms
- **Стало:** 7000-10000ms
- Чертежи живут дольше

### 3. Использован `while` вместо `if` ✅
```javascript
// Было:
if (this.activeBlueprints.length < this.maxBlueprints) {
    this.createBlueprint(randomType);
}

// Стало:
while (this.activeBlueprints.length < this.maxBlueprints) {
    this.createBlueprint(randomType);
}
```
- Создаются все необходимые чертежи сразу

### 4. При старте создаются 5 чертежей ✅
```javascript
createFirstBlueprint() {
    // Создаем сразу 5 чертежей при старте
    setTimeout(() => this.createBlueprint('CylindricalPot'), 100);
    setTimeout(() => this.createBlueprint('SquarePot'), 200);
    setTimeout(() => this.createBlueprint('RoundPot'), 300);
    setTimeout(() => this.createBlueprint('TaperedPot'), 400);
    setTimeout(() => this.createBlueprint('TallPot'), 500);
}
```
