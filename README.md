# 🏭 SANTINO - Премиум пластик

**Чистая продакшен версия сайта завода пластиковых изделий**

## 📋 Описание проекта

SANTINO - это современный веб-сайт для завода премиальных пластиковых изделий. Проект представляет собой одностраничное приложение с интерактивными элементами, анимациями и адаптивным дизайном.

## 🚀 Особенности

- **Современный дизайн** - Минималистичный и элегантный интерфейс
- **Адаптивность** - Полная поддержка всех устройств
- **Производительность** - Оптимизированная загрузка и быстрая работа
- **Интерактивность** - Плавные анимации и эффекты
- **Фоновые чертежи** - Профессиональные технические чертежи горшков
- **Модульная архитектура** - Чистая структура кода
- **SEO-оптимизация** - Правильная структура для поисковых систем

## 🛠 Технологии

### Frontend
- **HTML5** - Семантическая разметка
- **CSS3** - Современные стили с переменными и Grid/Flexbox
- **JavaScript (ES6+)** - Модульная архитектура
- **GSAP** - Профессиональные анимации
- **Swiper.js** - Карусели и слайдеры

### Инструменты разработки
- **Vite** - Быстрая сборка и разработка
- **Git** - Контроль версий

## 🚀 Установка и запуск

### Предварительные требования
- Node.js (версия 20.19+ рекомендуется)
- npm или yarn

### Установка зависимостей
```bash
npm install
```

### Команды Vite (рекомендуется)
| Команда | Описание |
|---------|----------|
| `npm run dev` | Dev-сервер с HMR на порту 3001 |
| `npm run build` | Сборка в папку `dist/` |
| `npm run preview` | Просмотр собранного `dist/` локально |
| `npm run lint` | ESLint-проверка поддерживаемого JS-слоя |
| `npm run format:check` | Проверка форматирования Prettier |
| `npm run test:smoke` | Smoke E2E (Playwright, Chromium) |

### Quality Gate (локально перед merge)
1. `npm run lint`
2. `npm run format:check`
3. `npm run test:smoke`

Все три шага должны быть зелёными.

### Альтернативный запуск (Python)
Для статической раздачи без Vite: `npm run start` или `npm run serve` (порт 3001).

## 📱 Адаптивность

Проект полностью адаптивен и поддерживает:
- **Desktop** (1200px+)

## ⚡ Производительность

Проект оптимизирован для максимальной производительности:
- **Lazy Loading** - Ленивая загрузка изображений
- **Critical CSS** - Критический CSS встроен в HTML
- **Минификация** - Сжатие CSS и JavaScript
- **Кэширование** - Настройки кэширования для статических ресурсов

## 🚀 Развертывание

### На хостинге reg.ru
1. Соберите проект: `npm run build`
2. Загрузите содержимое папки `dist/` на сервер
3. Настройки Apache уже включены в `.htaccess`
