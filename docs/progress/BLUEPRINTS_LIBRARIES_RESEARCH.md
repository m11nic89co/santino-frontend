# 📐 Исследование библиотек для технических чертежей

## 🎯 Задача
Найти готовые библиотеки для создания технических чертежей в стиле AutoCAD/3D Max для фона сайта.

---

## 📚 Найденные библиотеки

### 1. **SVG.js** ⭐⭐⭐⭐⭐
**Описание:** Легковесная библиотека для работы с SVG

**Возможности:**
- ✅ Создание SVG элементов программно
- ✅ Математические операции с путями
- ✅ Анимации и трансформации
- ✅ Размер: ~25KB (gzipped)

**Использование:**
```javascript
import { SVG } from '@svgdotjs/svg.js'

const draw = SVG().addTo('#container').size(300, 300)
const line = draw.line(0, 0, 100, 100).stroke({ width: 2, color: '#fff' })
```

**Оценка:** Отлично подходит для простых чертежей

---

### 2. **Fabric.js** ⭐⭐⭐⭐
**Описание:** Canvas библиотека для интерактивной графики

**Возможности:**
- ✅ Работа с Canvas
- ✅ Интерактивные объекты
- ✅ Сериализация
- ⚠️ Размер: ~200KB

**Использование:**
```javascript
const canvas = new fabric.Canvas('canvas')
const line = new fabric.Line([0, 0, 100, 100], {
  stroke: '#fff',
  strokeWidth: 2
})
canvas.add(line)
```

**Оценка:** Слишком тяжелая для фоновых элементов

---

### 3. **Paper.js** ⭐⭐⭐⭐
**Описание:** Профессиональная векторная графика

**Возможности:**
- ✅ Математические операции с путями
- ✅ Анимации
- ✅ Интерактивность
- ⚠️ Размер: ~150KB

**Оценка:** Избыточно для фона

---

### 4. **Three.js** ⭐⭐⭐⭐⭐
**Описание:** 3D графика для веба

**Возможности:**
- ✅ 3D чертежи
- ✅ Камеры и перспектива
- ✅ Shadows и lighting
- ⚠️ Размер: ~600KB

**Использование:**
```javascript
const scene = new THREE.Scene()
const geometry = new THREE.CylinderGeometry(5, 5, 10, 32)
const material = new THREE.LineBasicMaterial({ color: 0xffffff })
const cylinder = new THREE.Line(geometry, material)
scene.add(cylinder)
```

**Оценка:** Самый профессиональный вариант, но тяжелый

## 🎯 Рекомендации для Santino

### Вариант 1: SVG.js (Рекомендован) ⭐⭐⭐⭐⭐
**Почему:**
- ✅ Легковесная (~25KB)
- ✅ Простой API
- ✅ Идеальна для технических чертежей
- ✅ Можно создать стиль AutoCAD
- ✅ Нет зависимостей

**Пример создания горшка:**
```javascript
import { SVG } from '@svgdotjs/svg.js'

function drawPot() {
  const draw = SVG().size(280, 340)
  
  // Боковые линии
  draw.line(60, 50, 140, 280)
      .stroke({ color: '#fff', width: 2.5 })
  
  draw.line(220, 50, 140, 280)
      .stroke({ color: '#fff', width: 2.5 })
  
  // Верхняя линия
  draw.line(60, 50, 220, 50)
      .stroke({ color: '#fff', width: 2.5 })
  
  // Окружность (вид сверху)
  draw.circle(70)
      .center(140, 120)
      .fill('none')
      .stroke({ color: '#fff', width: 2 })
  
  return draw
}
```

### Вариант 2: Three.js (Профессиональный) ⭐⭐⭐⭐
**Почему:**
- ✅ 3D чертежи выглядят максимально профессионально
- ✅ Реалистичная перспектива
- ✅ Можно вращать камеру
- ⚠️ Тяжелый (~600KB)
- ⚠️ Сложнее интеграция

**Пример:**
```javascript
import * as THREE from 'three'

const scene = new THREE.Scene()
const camera = new THREE.PerspectiveCamera(75, 1, 0.1, 1000)
const renderer = new THREE.WebGLRenderer({ alpha: true })

// Цилиндр (горшок)
const geometry = new THREE.CylinderGeometry(1, 0.8, 1.2, 32)
const edges = new THREE.EdgesGeometry(geometry)
const material = new THREE.LineBasicMaterial({ color: 0xffffff })
const pot = new THREE.LineSegments(edges, material)
scene.add(pot)

camera.position.z = 3
renderer.render(scene, camera)
```
