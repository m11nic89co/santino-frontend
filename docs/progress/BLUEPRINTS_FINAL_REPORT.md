## ✅ Созданные чертежи (9 типов):

1. **CylindricalPot** (Цилиндрический) - Ø180мм × 200мм

3. **RoundPot** (Круглый) - Ø200мм × 150мм
4. **TaperedPot** (Сужающийся) - Ø160мм × 220мм
5. **TallPot** (Высокий) - Ø140мм × 260мм

### Премиум горшки:

7. **PotWithStand** (С подставкой) - Ø160мм × 180мм с ножкой

9. **VasePot** (Ваза) - Ø180мм × 280мм высокая

## 🎯 Профессиональные элементы (ГОСТ):

### 1. ✅ Центровые линии (пунктир 2-1)
   - Выходят за контур на 3-5мм
   - Показывают центр симметрии
   - `stroke-dasharray: '3,2'`

### 2. ✅ Осевые линии (ось симметрии)
   - Горизонтальная и вертикальная
   - Двойная (толстая + тонкая пунктирная)

### 3. ✅ Точки отсчета
   - Круги на пересечениях линий
   - Размер 2px
   - Отметки для привязки размеров

### 4. ✅ Размеры со стрелками (ГОСТ)
   - Разрывные линии с зазором 15-20мм
   - Стрелки в 4 направлениях (left/right/top/bottom)
   - Размер 3-4px
   - Текст жирным моношрифтом

### 5. ✅ Штриховка сечений (наклонные)
   - Наклон 45°
   - Шаг 10-16мм
   - Показывает толщину стенок
   - `stroke-width: 0.6-0.8`

### 6. ✅ Пунктирные линии для проекций
   - ВИД СВЕРХУ
   - ВИД СБОКУ
   - ВИД СНИЗУ
   - `stroke-dasharray: '4,2'`

### 7. ✅ Шкала измерения (линейка)
   - Деления 0, 50, 100мм
   - Риски перпендикулярные
   - Подписи шкалы
   - Для контроля масштаба

### 8. ✅ Техническая информация (ГОСТ)
   - "ВИД СВЕРХУ (А-А)"
   - "ВИД СНИЗУ"
   - "ВИД СБОКУ"
   - Буквы A, B, C для разрезов
   - Размеры в мм жирным

### 9. ✅ Диагонали для 3D эффекта
   - Показывают объем
   - Соединяют проекции
   - Тонкие линии 1-1.5px

### 10. ✅ Внутренние контуры
    - Толщина стенок (5мм)
    - Объемные линии
    - Контур дна отдельно

## 🎨 Особенности каждого чертежа:

### CylindricalPot:
- 3 окружности (вид сверху, снизу, осевая)
- Осевая линия пунктиром
- Дно горшка выделено
- Точки измерений
- ГОСТ-подписи

### SquarePot:
- Квадратные проекции
- Диагонали для 3D эффекта
- Двойной контур
- Штриховка стенок

### RoundPot:
- Двойные окружности
- Радиусные размеры со стрелками
- Заполненное кольцо
- Боковые стенки

### TaperedPot:
- Трапеция (сужение)
- Две окружности (верх/низ)
- Вспомогательные линии сужения
- Внутренние линии объема

### TallPot:
- Высокий узкий контур
- Двойные окружности
- Перпендикуляры для размеров
- Горизонтальные и вертикальные размеры

### DecorativePot:
- Орнаментные линии
- Декоративные окружности
- Сложная форма с узором
- Точки крепления

### PotWithStand:
- Подставка (ножка)
- Двойной контур горшка
- Основание подставки
- Отдельные размеры подставки

### HangingPot:
- Подвес (веревка/цепь)
- Точки крепления
- Кашпо (без дна)
- Внутренний контур

### VasePot:
- Высокое горло
- Широкое дно
- Орнаментные линии
- Декоративные элементы
